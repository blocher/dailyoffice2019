Poetry Indentation Fix - Isaiah 65:1-2

❌ BEFORE (Regular Spaces - Attempt 2)

HTML Code:

<span class="indent-1-breaks"> </span> (4 regular spaces)

Rendered Output:

1I was ready to be sought by those who did not ask for me;
I was ready to be found by those who did not seek me.
I said, "Here I am, here I am,"
to a nation that was not called by my name.

Problem: Regular spaces don't create visible indentation (collapsed by HTML)

✅ AFTER (HTML &nbsp; Entities - Attempt 3)

HTML Code:

<span class="indent-1-breaks">&nbsp;&nbsp;&nbsp;&nbsp;</span> (4 non-breaking spaces)

Rendered Output:

1I was ready to be sought by those who did not ask for me;
    I was ready to be found by those who did not seek me.
I said, "Here I am, here I am,"
    to a nation that was not called by my name.
2I spread out my hands all the day
    to a rebellious people,
who walk in a way that is not good,
    following their own devices;

Solution: &nbsp; entities create actual visible indentation that renders correctly

Technical Details

Code Change:

# BEFORE (Attempt 2 - Regular spaces)
html_parts.append('<span class="indent-1-breaks">    </span>')

# AFTER (Attempt 3 - HTML entities)
html_parts.append('<span class="indent-1-breaks">&nbsp;&nbsp;&nbsp;&nbsp;</span>')

Why This Works:

  • Regular spaces are collapsed by HTML rendering engines
  • &nbsp; (non-breaking space) entities are preserved and displayed
  • 4× &nbsp; = visible 4-space indentation
  • 8× &nbsp; = visible 8-space indentation (for deeper nesting)
  • Works without any CSS styling required
